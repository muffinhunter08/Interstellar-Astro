# Interstellar V7

Web proxy with multi-tab browser interface.

## Deployment

```bash
git clone https://github.com/UseInterstellar/Interstellar-Astro
cd Interstellar-Astro
npm install && npm start
```

## Development

```bash
npm install && npm run dev
```

## Tech Stack

- Astro 5 + React 19
- Fastify backend
- Scramjet proxy engine
- Tailwind CSS
